Sincronización entre dispositivos (opcional)
Configura GitHub Gist para que tus casos se sincronicen entre PC y celular. ¿Cómo obtener estos datos?
Paso 1: Ve a github.com → tu avatar → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token (classic). Selecciona solo el scope gist. Copia el token.
Paso 2: Ve a gist.github.com → New gist. Nombre del archivo: sapdx_data.json. Contenido: {}. Selecciona Secret. Copia el ID de la URL (los 32 caracteres después de gist.github.com/tuusuario/).
Sin Gist: los datos se guardan solo en este dispositivo · Con Gist: sincronizado entre todos tus dispositivos
